## Avançado – C/C++

Exercícios avançados desenvolvidos em C/C++ utilizando múltiplos conceitos
dentro de um mesmo código.

### Conceitos aplicados
- Structs
- Funções
- Laços de repetição (for, do..while)
- Manipulação de dados
- Organização lógica do código

### Observação
Os exercícios foram desenvolvidos como soluções completas, reunindo vários
conceitos no mesmo arquivo, simulando situações mais próximas de aplicações reais.
